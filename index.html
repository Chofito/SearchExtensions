SearchExtensions

Library of IQueryable and IEnumerable extension methods to perform searching

Home Simple Search String Search Soundex Levenshtein Release History

This project is avaliable for download as a nuget package

PM> Install-Package NinjaNye.SearchExtensions

Search Extensions provides a collection of extension methods to IQueryable and IEnumerable that enable easy searching and ranking. Searches can be performed against multiple properties and support a wide range of types

Examples

For examples and more detailed explanations, please see the following sections:

  • Simple Searching - Provides the ability to search multiple primitive properties
  • String Searching - Provies fast, advanced string searching between multiple properties and against multiple search terms.
  • Soundex Search - Provides the ability to search multiple properties for values that match the soundex value of another string
  • Levenshtein Search - Provides the ability to calculate the Levenshtein distance of each property against a particular term or other property.
